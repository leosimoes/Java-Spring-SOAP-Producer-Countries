## Development steps
The steps of project implementation:

1. Create project (in IntelliJ) with:
- Java language (17);
- Spring Framework (6.2.3);
- Dependencies: Web, Web Services, DevTools and Lombok;
- Plugins: Lombok and Jakarta EE: Web-Services (JAX-WS).
